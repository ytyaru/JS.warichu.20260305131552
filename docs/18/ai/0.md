# 割注

日本語の小説などで簡易記法による割注をHTMLで表現したい。

青空文庫の資料に割注の画像があるので理解のため参考にする。
https://www.aozora.gr.jp/annotation/etc.html#warichu

# 要件

* メタ文字: `〔`,`｜`,`〕`
* `〔ここにあるテキストを二行に分割して表示する〕`
* 縦書き／横書き両用
* 標準文字サイズの半分で二行に渡り表記される
	* 各行に対して配置位置を指定できる（`start`,`end`,`center`,`justify`）
* プレフィックスとサフィックスの指定（任意で括弧を表示できる）
	* 全角括弧を半角幅で表示するか否か（CSS）
	* テキストコピー是非を指定できる
* 行分割位置の指定（手動と自動を選択可能）
	1. `｜`がある箇所で行分割する（片方の字数がゼロなら例外発生）
	2. `｜`がないならIntl.Segmenterの'word'単位で分割し、行字差が`option.diff`(デフォルト`1`)以内の箇所があれば分割する
	3. 上記いずれにも該当しない場合、中点で分割する（IEEE754誤差のないよう計算し全字が奇数なら一行目を長くする）
* 異常発生時の対処方法（例外発生＆中断／警告表示＆継続／任意例外クラス／任意関数（`true`を返すと中断））

```markdown
〔ここにあるテキストを二行に分割して表示する〕
```

ライブラリとして公開する。次のように使う。

```html
<script src="warichu.js"></script>
<script>document.body.innerHTML = ((new Warichu()).parse('〔これは割注です〕'));</script>
```

bun,node,browserのような異なる文脈でも使えるようにする。

## 1. 実行環境

* language
	* TS
	* JS
	* CSS
* target
	* bun（TS有）
	* browser（TS無）
	* node（TS無）
* format
	* esm（`<script type="module">`, `export {...}`）
	* no-esm（esmからexport構文を削除したもの  `<script>`）
	* iife（即時実行関数 `<script>`, `(function(){...})();`）
* release
	* debug（開発環境用。非ミニファイ。console系API全残し）
	* code（共用環境用。非ミニファイ。console.info/warn/error以外削除）
	* min（本番環境用。ミニファイ。console.info/warn/error以外削除）

## 2. ファイル

実行環境に応じたファイルを30個作成する。

```
27 = target(bun,browser,node) * format(esm,no-esm,iife) * release(debug,code,min)

+ bun/ts
+ css/code
+ css/min

= 30
```

TSはbun環境でしか使えない。CSSにはdebugはないため不要。以上のことから3ファイルだけを基本27ファイルに追加した合計30ファイルを作成する。

このファイルは作成したTS実装コードをベースにして、bun buildコマンドで生成する。

## 5. オプション

```ts
export type ErrorBehavior = 
  | 'throw' 
  | 'warn' 
  | { new (message: string): Error } 
  | ((message: string) => boolean | void);
export interface WarichuOptionSign : {
  str?: string;        // 前後の括弧記号（例: '（）'）
  copyable?: boolean;  // 括弧をコピー可能にするか（デフォルト:true）
  narrow?: boolean;    // 全角括弧を半角幅にするか （デフォルト:false）
},
export interface WarichuOptionSplit: {
  diff?: number;     // 自動分割で許容する最大字数差（デフォルト1。0〜Nの正数値）
  priority?: number; // 等分不能な時一行目と二行目のどちらを長くするか（デフォルト1。1か2）
}
export interface WarichuOptionArign: {
  start?: string;      // 一行目の揃え（デフォルト: start。end,center,justify）
  end?: string;      // 二行目の揃え（デフォルト: start。end,center,justify）
}
export interface WarichuOption {
  sign? : WarichuOptionSign;   // 割注の前後にある記号
  split? : WarichuOptionSplit; // 割注のテキスト分割位置
  align? : WarichuOptionArign; // 割注のテキスト配置
  error?: ErrorBehavior;       // エラー時の挙動（デフォルト:'throw'）
}
```

## 6. 例外・警告

* オプション、引数(text)
	* `型が不正です。対象キー:${key} 期待値:${expected} 実際値:${actual}`
	* `値が不正です。対象キー:${key} 期待値:${expected} 実際値:${actual}`
* 割注
	* `割注は合計2文字以上必要です: "${content}"`
	* `割注の片方の行が空です。: start:${start}, end:${end}`
	* `割注にパイプ｜が複数あります。一個まで有効です。`
