# テスト観点

## 0. テスト対象クラス抽出

* 27種類のdist/ファイル全てに対して`Warichu`クラスを取得し返すこと
* もし返せないならテスト失敗

## 1. WarichuErrorクラス

* `WarichuError`という名前のError継承クラスが存在すること
* コンストラクタ引数が`(message, cause)`であること(`super(message, cause ? {cause:cause} : undefined)`)
* 指定したメッセージが表示されること
* 指定した原因エラーが表示されること

## 2. WarichuOptionオブジェクト

* デフォルト値確認
	* 想定する全キーが存在すること
	* 型
		* 各キーに想定内型データをセット可能なこと
		* 各キーに想定外型データをセットしたら例外発生すること
	* 値
		* 各キーに想定内値をセット可能なこと
		* 各キーに想定外値をセットしたら例外発生すること

## 3. Warichuクラス

* `Warichu`クラスが存在すること
* `Warichu`コンストラクタ
	* 引数なし
		* デフォルト・オプションが使用されること
	* 引数あり
		* 非オプション型: 例外発生
		* オプション型
			* オプション値のチェック処理をすること（`2. WarichuOptionオブジェクト`）
				* 指定されたオプション値がセットされること
				* 未指定オプションはデフォルト値がセットされること
* `getGraphemes`
	* 書記素数が想定通りであること
		* サロゲートペアなし
		* サロゲートペアあり
		* 合字あり
* `_handleError`
	* `'throw'`
		* `WarichuError`クラス型であること
		* 指定されたメッセージであること
		* 処理が中断されること
	* `'warn'`
		* `console.warn`で指定メッセージが出力されること
		* 処理が続行されること
	* `function`
	* `Errorクラス`
	* 上記以外
		* TypeErrorを発生すること
		* `option.errorは'throw','warn',任意Errorクラス,任意関数のいずれかであるべきです。`というメッセージであること
* `_getSplitPoint`
	* 2文字未満
		* 例外発生: WarichuError
		* メッセージ: `割注は合計2文字以上必要です: "${content}"`
	* 手動分割`｜`
		* `｜`があれば
			* その位置で分割すること
			* 但し片方の行の字数がゼロなら
				* 例外発生: WarichuError
				* メッセージ:`"割注の片方の行が空です。"`
		* `｜`がなければ以降へ
	* 自動分割(Intl.Segmenter 'word')
		* `option.diff`以内の字差で済む箇所を全箇所取得する
			* このうち字差が最も少なく済む箇所を取得する
				* 2箇所の場合
					* `option.priority`(一行目を長くするか／二行目を長くするか)
						* `option.priority=1`なら一行目が長くなる位置で分割する
						* `option.priority=2`なら一行目が長くなる位置で分割する
				* 1箇所の場合
					* その箇所で分割する
				* 0箇所の場合
					* 以降へ
	* 中間分割
		* 偶数(2,4,6,8,10,40,80,400)
			* 丁度中間位置で分割されること
		* 奇数(3,5,7,9,33,99,999)
			* 一行目が一文字長く分割されること
* `parse(text, option)`
	* もしoptionが
		* あればコンストラクタで指定したオプションでマージした合成オプションを使うこと
			* オプション値のチェック処理をすること（`2. WarichuOptionオブジェクト`）
		* なければコンストラクタで指定したオプションで実行すること
	* パイプが複数ある
		* 例外発生: WarichuError
		* メッセージ: `パイプ(｜)が複数あります。無いか一個のみ許容します。`
	* オプション設定に応じたHTMLであること
		* `sign`
			* 記号は空文字か2文字のみ許可する
				* 空文字
					* 要素を出力しない
				* 2文字
					* 要素を出力する
			* それ以外
				* 例外発生: WarichuError
				* メッセージ: `signが不正です。空文字か2文字のみ有効です。: ${sing}`
		* `copyable`
			* `sign`要素の属性に`' aria-hidden="true" style="user-select:none;"'`を付与
				* `true`: する
				* `false`: しない 
		* `narrowSign`
			* class属性値に`is-narrow`を付与
				* `true`: する
				* `false`: しない
	* 所定のHTML構造であること
		* 要素の親子関係、順序
		* タグ名: `span`
		* class属性名: `warichu-line1`, `warichu-line2`
